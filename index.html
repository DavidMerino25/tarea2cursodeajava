Uso y significado de 5 palabras reservadas en Java

Las palabras reservadas son palabras que tienen un significado especial para el compilador de Java. Por ejemplo, la palabra reservada class se utiliza para definir una clase. La palabra reservada public se utiliza para definir un método o una clase como pública. La palabra reservada static se utiliza para definir un método como estático. La palabra reservada void se utiliza para definir un método que no devuelve ningún valor. La palabra reservada return se utiliza para devolver un valor de un método.

static

La palabra reservada static Se utiliza para declarar una variable, método o clase interna como un variables de clase. Las clases mantienen una copia de lass variables de clase independientemente de cuántas instancias existen de esa clase. Static además se usa para definir un método como un método de clase. Los métodos de clase están vinculados a la clase en lugar de a una instancia específica, y solo pueden operar en campos de clase. ¿Cómo compilar un archivo .java y como ejecutarlo usando solamente la terminal de comandos (Terminal/CMD).?

Para compilar un archivo .java se debe usar el comando javac seguido del nombre del archivo .java que se desea compilar. Por ejemplo, si se desea compilar el archivo "HolaMundo.java" se debe usar el comando javac HolaMundo.java. Para ejecutar un archivo .class se debe usar el comando java seguido del nombre del archivo .class que se desea ejecutar. Por ejemplo, si se desea ejecutar el archivo "HolaMundo.class" se debe usar el comando java HolaMundo
